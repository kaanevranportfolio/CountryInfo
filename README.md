# CountryInfo

A React application that displays information about countries, including their flags, population, region, and capital. Users can search for countries, filter by region, and view detailed information about a specific country.

## Features

- **[Country List](#country-list)**: Displays a list of countries with their flags, population, region, and capital.
- **[Search Functionality](#search-functionality)**: Allows users to search for countries by name.
- **[Region Filter](#region-filter)**: Enables filtering countries by region.
- **[Pagination](#pagination)**: Implements pagination to show a limited number of countries per page.
- **[Dark Mode](#dark-mode)**: Toggles between light and dark themes for better user experience.
- **[Country Details](#country-details)**: Provides detailed information about a selected country, including currencies and languages.
- **[Responsive Design](#responsive-design)**: Ensures the application is usable on various screen sizes.
